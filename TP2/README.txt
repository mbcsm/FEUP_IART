To run the project simply load it into your prefered IDE, we used intellij during the developement proccess.

If you want to change the Board used or any of the search values change the variables on top of the Main.java class, these include:
private static int DLS_MAX_SEARCH_DISTANCE;
private static int A_VERTICAL_COST;
private static int A_HORIZONTAL_COST;
private static String boardName;
